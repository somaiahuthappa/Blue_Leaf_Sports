Instructions

Please follow the following instruction for each section, to retrieve the desired information.

  • League Teams: Please include the name of the country and the name of the league you want to recieve back. For both soccer and hockey leagues/cups.
  • Team Players: Please include the full name of the team if the shorthand name does not work. Only for soccer teams.
  • League/Cup Top Scorers: Please include the name of the country and the name of the league you want to view its top scorers. Only for soccer leagues/cups.
